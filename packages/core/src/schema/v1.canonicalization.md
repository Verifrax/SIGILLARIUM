# Sigillarium Bundle Canonicalization — v1 (FROZEN)

Rules (must not change within v1):
- Deterministic key ordering: lexicographic by UTF-16 code unit.
- No locale-dependent transforms.
- No timestamps, mtimes, or environment-derived fields.
- UTF-8 encoding; LF line endings.
- Paths normalized to POSIX `/`.
- Sorting is stable; ties resolved by full byte comparison.

Any rule change requires a new schema version.
